1. Project Description:
	The purpose of this project is to track customer's daily diet and calorie intake. The frontend is implemented with React, and backend is by NodeJS/Mongoose.
		
2. How to run:
	1) Run MongoDB:
		mongod --dbpath "your MongoDB path"
	2) run server:
		npm run server
	3) run client:
		npm run client
	4) compile less into css file:
		gulp dev
	5) run eslint:
		npm run lint
